Welcome to the Directory of Forest Products Industries

The Directory of Forest Products Industries is an interactive application developed and maintained by Texas A&M Forest Service. The application includes a suite of tools that allows users to find information about wood-manufacturing industries through a spatial search using mapping tools or a text search with drop-down menus. The results of both searches are displayed on a map. Additional tools allow users to preview and print the map and to save the results to PDF or excel files.
